# Requirements

* beanstalkd
* python3
* pyyaml
* pystalkd

# Get started

1. Clone the repo.

2. Install dependencies

```
apt-get install beanstalkd
virtualenv -p /usr/bin/python3 ~/.virtualenvs/lab-pystalkd
source ~/.virtualenvs/lab-pystalkd/bin/activate
pip install pyyaml pystalkd
```

3. Put and get jobs.
